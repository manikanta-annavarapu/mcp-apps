## How to referesh in ChatGPT App

In the setting when you connect the app you will see a refresh button like in the screenshot below.

In the dev mode, this is useful, when you stopped the server and built a new tool and start the server back again, then the new tool is not available in the chatGPT, it will be only available when you hit available.

![Refresh Button Screenshot](/docs/screenshots/RefreshButtonScreenshot.png)

You can also see list of all tools, if you scroll down
